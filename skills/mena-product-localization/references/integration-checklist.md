# Product Localization Integration Checklist

## Answer Shape

- Files read.
- Source-backed facts.
- Recommendations.
- Unknowns.
- Validation steps.
- Production launch gate.

## Review Steps

- Confirm locale, country, audience, channel, and surface.
- Check RTL, bidi, Arabic copy, numerals, dates, phones, addresses, currency, VAT labels, checkout, and accessibility.
- Route high-risk implementation to the owning skill.

## Production launch gate

- Use sandbox/test data before production.
- Protect secrets, credentials, phone numbers, addresses, IDs, and payment references.
- Require explicit approval before publishing, sending, launching, or changing production checkout/payment/localization behavior.
