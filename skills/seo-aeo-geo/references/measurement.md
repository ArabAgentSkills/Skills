# Measurement

Use this when the user asks how to prove SEO, AEO, or GEO work is improving visibility.

## Inputs

- Google Search Console performance and indexing reports.
- Sitemap and robots outputs from the production domain.
- Server, CDN, or Netlify logs where available.
- Build output, route inventory, and generated public data files.
- Manual AI-search prompt audits with saved prompts and dates.

## KPIs

- Intended URLs indexed.
- Crawl errors reduced.
- Search impressions and non-brand clicks.
- Query coverage for skill/category/vendor/topic terms.
- Source page visits and install/contribute clicks.
- AI crawler access success rates.
- AI answer citations or mentions, recorded as directional evidence only.

## Caution

Do not report prompt-audit observations as a stable ranking metric. AI answers vary by model, time, user context, and retrieval path.
