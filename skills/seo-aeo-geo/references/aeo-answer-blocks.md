# AEO Answer Blocks

Use this when creating content meant to answer a specific question clearly.

## Good Answer Block

- Starts with the direct answer in one or two sentences.
- Names the scope: country, audience, channel, category, or skill family.
- Links to the relevant source-backed page.
- States uncertainty or review status when facts are incomplete.
- Avoids hype, broad claims, and unsupported superlatives.

## Template

```md
## Short Answer

<Direct answer>. Use this for <audience/task/scope>. Source confidence: <low|medium|high>. Review status: <status>.

Next reads:
- <canonical internal page>
- <source transparency page>
- <related skill or vendor>
```

## Do Not

- Add answer blocks that are disconnected from page content.
- Turn uncertain claims into FAQs.
- Hide disclaimers below the fold when risk is high.
- Create multiple near-duplicate answer pages for the same intent.
