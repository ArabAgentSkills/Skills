# B2B Packaging

## Self-Serve SME

- Public prices can work when implementation is light and value can be reached without sales help.
- Use clear plan limits: users, branches, invoices, messages, channels, modules, API access, integrations, support, storage, or AI credits.
- Keep entry plans useful, not fake.
- Free trial only works when activation can happen quickly without live regulated data.

## Assisted SMB And Mid-Market

- Use starting-from prices or tier bands only if sales can honor them.
- Separate setup/onboarding, recurring subscription, usage fees, and add-ons.
- Include migration, training, support, implementation timeline, and success owner assumptions.

## Enterprise And Regulated Buyers

- Prefer request-demo, contact-sales, or request-quote when cost depends on branches, employees, integrations, usage, compliance, SLA, migration, or security review.
- Include procurement-ready fields: legal entity, tax registration, quote validity, payment terms, SLA, implementation scope, security notes, renewal terms, and cancellation terms.

## Confidence

Packaging patterns can be medium or high confidence. Exact prices and thresholds are low confidence unless backed by current customer data or current public sources.
