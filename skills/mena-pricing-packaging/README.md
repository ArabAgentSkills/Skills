# MENA Pricing Packaging

This skill helps agents design and review MENA pricing pages, SaaS packaging, ecommerce offers, discounts, VAT/currency display, BNPL/installment language, and enterprise quote structures.

## Main Uses

- Pricing-page localization and plan QA.
- Self-serve vs demo-first package recommendations.
- VAT and currency display review.
- BNPL, installments, annual savings, and discount safety.
- Enterprise and procurement-friendly quote structures.

## Maintenance

- Keep `SKILL.md` short and procedural.
- Put detailed patterns and caveats in `references/*.md`.
- Keep exact price claims out unless they are source-backed snapshots with checked dates.
- Run `pnpm skills:validate`, `pnpm validate`, and relevant checks after edits.

## Boundaries

This skill drafts and reviews. It does not change live prices, send quotes, submit tenders, enable payment methods, or provide tax/legal advice.
