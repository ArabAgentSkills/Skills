# Source-Backed Answer Example

Request: Compare growth channels for a country-specific launch.

## Files read

- `skills/mena-growth-channels/SKILL.md`
- `skills/mena-growth-channels/sources.yml`
- The matching `references/*.md` file for each channel.
- `skills/mena-growth-channels/references/integration-checklist.md` for launch readiness.

## Recommendation

Recommend a small set of testable channels, not a full budget allocation, unless current country, segment, channel, and performance evidence is provided.

## Source-backed notes

Each channel note must include country, segment, channel, confidence, evidence, source URL, and review status.

## Unknowns

List missing keyword data, audience assumptions, creative review, platform review, local legal/privacy review, marketplace policy, partner access, tracking state, and operations gaps as `Unknown from public docs`, `Unknown from public sources`, `Needs vendor access`, or `needs expert review`.

## Validation steps

Include sandbox/test checks, landing-page QA, consent/privacy checks, policy review, budget cap, stop rule, rollback notes, and a production approval gate.

Avoid:

- Inventing channel performance or CAC claims.
- Treating public statistics as demand proof.
- Launching, sending, publishing, uploading audiences, or committing commercially without explicit approval.
