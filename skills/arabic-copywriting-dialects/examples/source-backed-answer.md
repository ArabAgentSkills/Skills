## Recommendation

Use MSA for cross-market transactional copy. Draft dialect variants only for native review and do not publish them until a reviewer approves the exact country, audience, channel, and use case.

## Unknowns

- Target country for any dialect version.
- Approved bilingual glossary.
- Platform/account requirements for production send.
- Native reviewer scope and approval status.

## Validation steps

- Label all dialect output `draft/native_review_required`.
- Check numbers, URLs, codes, names, dates, and punctuation in RTL preview.
- Screenshot the final surface on mobile and desktop where applicable.
- Get explicit approval before send, launch, publish, or production release.
