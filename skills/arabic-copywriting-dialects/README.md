# Arabic Copywriting And Dialects

Use this skill to draft, localize, and review Arabic copy while keeping dialect output clearly gated for native review.

## Scope

- MSA vs dialect selection.
- Egyptian, Gulf/Saudi, Levantine, and Maghrebi routing notes.
- WhatsApp, SMS, landing-page, support, and mixed Arabic-English copy QA.
- RTL, locale, bidi, and native-review checks.

## Operating Principle

This skill creates review-ready drafts and QA notes. It does not certify dialect perfection, local legal compliance, platform approval, or cultural acceptability.

## Usage

Point an agent to `skills/arabic-copywriting-dialects/SKILL.md`, then load only the reference file that matches the requested market or channel.

## Maintenance

- Keep source-backed facts in `sources.yml` and references.
- Mark all dialect examples as `draft/native_review_required` unless a scoped native review exists.
- Run regional and repo validation after edits.
