# Integration Checklist

This compatibility checklist maps the regional copy skill to the repo's current skill validator. Prefer the specific reference files for actual copy guidance.

## Answer Shape

- Files read.
- Scope: market, audience, channel, language variety, intended use, and risk tier.
- Recommendation.
- Draft or review output with status labels.
- Unknowns and review gates.
- RTL/channel QA.
- Validation steps.

## Production launch gate

- No outbound message, paid ad, public publish, support macro deployment, or production UI release without explicit approval.
- Use sandbox, preview, or staging checks before production.
- Confirm opt-in, platform policy, and local compliance for WhatsApp, SMS, email, push, sales, and ads.
- Require native review for dialect and expert review for regulated or sensitive claims.
