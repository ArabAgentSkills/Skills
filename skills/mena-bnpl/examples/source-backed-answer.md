# Source-Backed Answer Example

Request: Compare two vendors for a country-specific integration.

Good answer shape:

1. State which skill and vendor files were read.
2. Compare only facts present in vendor files and source URLs.
3. Separate public docs from gated/partner-only docs.
4. List unknowns that require vendor access.
5. Provide implementation and validation steps.

Avoid:

- Inventing endpoint paths.
- Claiming support for flows not in source-backed notes.
- Treating a blog or directory as ground truth.
