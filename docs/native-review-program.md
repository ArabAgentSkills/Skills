# Native Review Program

Native review is for language quality, local tone, dialect fit, idioms, public brand copy, customer-facing support, and sensitive Arabic-English wording.

## What Reviewers Check

- Language or dialect label is correct.
- The copy sounds natural for the stated country, audience, channel, and risk level.
- The copy avoids stereotypes, fake certainty, insensitive religious/cultural framing, and manipulative pressure.
- Public copy is marked approved only when the reviewer is comfortable with the exact scope.

## Public Data Rules

- Store only aggregate review status and public notes in public files.
- Do not store reviewer names, phone numbers, email addresses, private survey responses, IDs, or raw customer transcripts.
- Use `private_notes_path` only for internal evidence that is not exported publicly.
