# Roadmap

## 0.1

- Data-first registry.
- Eleven initial skills.
- Local heuristic evals.
- Website and GitHub Actions.
- Netlify-ready static website.
- Public skills export artifact.

## 0.2

- Deeper official docs crawling.
- OpenAPI/Postman extraction.
- Source drift PR automation.
- More country-specific coverage.
- Approval-gated submissions for new skills from users, companies, agencies, and organizations.
- First non-API skill proposals for Arab-market sales, marketing, localization, support, and content workflows.

## 0.3

- Weekly Codex maintenance automation that refreshes sources, regenerates skills, runs evals, and pushes reviewable updates.
- More examples per vendor.
- Contributor review dashboard.
- Maintainer workflow for approving or rejecting submitted skills before public publication.

## 1.0

- Stable P0 skills.
- Maintained weekly source checks.
- Clear release process.
- Broader Arab-world skill taxonomy beyond APIs, with source-backed guidance for business, culture, language, and go-to-market workflows.
